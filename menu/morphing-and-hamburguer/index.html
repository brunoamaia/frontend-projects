Projeto com animação

Páginas criadas com base no projeto: "Desenvolvendo um menu hamburguer e morphing menu com CSS Transitions", da Digital Inovation One.

O projeto foi dividido em 5 páginas. As 4 primeiras são para a criação de funcionalidades de modo isolado. E a ultima página consiste na criagão de uma Landing Page que utiliza quase todas as funcionalidades desenvolvidas anteriormente.


As páginas podem ser acessadas clicando nos links a seguir:


  • Card: um card com animalções ao passar o mouse em cima do quadro.
  • Chatbot loading: animação para representar que alguem está digitando.
  • Hamburguer: menu animado de três pontos para um "X" (para mobile).
  • Keyframe: plano de fundo animado e alguns componentes.
  • Projeto final: landing page utilizando três dos componentes descritos anteriormente.
